\section{文档类}

\begin{itemize}
  \item article，文章格式，常用于科技论文、报告、说明文档等。
  \item proc，基于article文档类的一个简单的学术文档模版。
  \item report，长篇报告，具有章节结构，用于综述、长篇论文、简单书籍等。
  \item book，书籍，包含章节结构、前言、正文、后记等。
  \item beamer，幻灯片
  \item minimal，精简文档，只设定了纸张大小和基本字号。
  \item standalone，得到大小合适的单页PDF，用于导出tikz绘图或公式。
\end{itemize}
