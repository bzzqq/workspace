\section{自动筛选}


\cepubblue{筛选区域或表中的数据：}
\begin{enumerate}
    \item 选择区域内的任意单元格。
    \item 选择“数据>筛选器”。
    \item 选择列标题箭头
    \item 选择“文本筛选器”或“数字筛选器”，然后选择条件
    \item 输入筛选条件，然后选择“确定”。
\end{enumerate}

\cepubblue{清除或删除筛选器：}
\begin{itemize}
    \item 清除列中的筛选器
    
    单击列标题“筛选”按钮，然后单击列名称中的“清除”。
    \item 清除所有筛选器
    
    单击“数据”选项卡，然后单击“清除”。
    \item 删除所有筛选器
    
    单击“数据”选项卡，然后单击“筛选”按钮
\end{itemize}

\section{高级筛选}

如果要筛选的数据需要复杂条件（例如类型 = “农产品”或销售人员 = “李小明”），可使用“高级筛选”对话框。

若要打开 “高级筛选” 对话框，请单击 “数据> 高级”。

可以在工作表以及要筛选的单元格区域或表格上的单独条件区域中键入高级条件。

\cred{使用等号键入文本或值}

由于在单元格中键入文本或值时等号 (=) 用来表示一个公式，因此 Excel 会计算键入的内容；不过，这可能会产生意外的筛选结果。 为了表示文本或值的相等比较运算符，应在条件区域的相应单元格中键入作为字符串表达式的条件：

=“= 条目”

其中\emph{条目}是要查找的文本或值。

\cepubblue{多列中有多个条件，所有条件都为真}

例：(类型=“农产品” AND 销售人员=“林恪”)

若要查找满足多列中有多个条件的行，请在条件区域的同一行中键入所有条件。

\cepubblue{多列中有多个条件，任何条件为真}

例：(类型=“农产品” OR 销售人员=“林恪”)

若要查找满足多列中有多个条件，其中任何条件都可以为真的行，请在条件区域的不同列和行中键入条件。

\cepubblue{多组条件，所有组集中于一列中}

例：( (销售额 > 6000 AND 销售额 < 6500 ) OR (销售额 < 500) )

若要查找满足多组条件，每组包含用于一列的条件的行，请在多个列中包括同一个列标题。

\cepubblue{多组条件，每组包含多列}

例： ( (销售人员 = “赵强” AND 销售额 >3000) OR (销售人员 = “林恪” AND 销售额 > 1500) )

若要查找满足多组条件，其中每组包括用于多列的条件的行，请在单独的列和行中键入每组条件。
