\begin{frame}
    \frametitle{学科定义}
    化学反应工程是化学工程学科的一个分支，通常简称为反应工程。

    反应工程是以工业反应过程为主要研究对象，以反应技术的开发、反应过程的优化和反应器设计为主要目的一门工程学科。它是在化工热力学、反应动力学、传递过程理论以及化工单元操作的基础上发展起来的。
\end{frame}

\begin{frame}
    \frametitle{化学反应工程的研究内容}
    \begin{enumerate}
      \item 反应动力学（三传一反）
            \begin{itemize}
                \item 反应的机理和速率
                \item 温度/浓度/压力对反应速率的影响
            \end{itemize}
      \item 反应器设计与分析
            \begin{itemize}
                \item 选择合适的反应器型式
                \item 选择合适的反应器操作方式
                \item 确定最佳的操作条件
                \item 计算完成规定的生产任务所需的反应体积
                \item 反应器优化
                \item[$\ast$] 反应器放大
            \end{itemize}
    \end{enumerate}
\end{frame}

\section{工业反应器的放大}


\begin{frame}
    \frametitle{反应放大}
        在实验室中用一个500 mL的烧瓶作为反应器对某吸热反应进行研究，物料通过反应器壁面与油浴进行换热以进行换热以满足反应的供热需求。现期望反应量增大1000倍，将反应器半径扩大到10倍，是否可行？
    \pause
    \[
        \begin{aligned}
            \text{反应体积扩大至：} \frac{\frac{4}{3}\pi R^3}{\frac{4}{3}\pi r^3} &= 1000 \ \text{倍} \\
            \text{传热面积扩大至：} \frac{4 \pi R^2}{4 \pi r^2} &= 100 \ \text{倍}
        \end{aligned}
    \]
    \pause
    \begin{itemize}
      \item 反应体积增大1000倍，但传热面积只能扩大100倍，使热量传递受到影响，从而可能影响反应进行的温度，反应就很难按预计的条件进行。
      \item 另一种情况，若反应是放热反应，传热面积小而不能有效移走反应热，导致反应系统热量积累使反应物系温度越来越高、反应速率越来越快，不仅影响产品的质量，甚至可能产生热爆炸而导致严重的安全问题。
    \end{itemize}
\end{frame}


\begin{frame}
    \frametitle{从实验室到工业生产}
    \begin{figure}[htpb]
        \centering
        \includegraphics[width=.45\linewidth]{放大}
    \end{figure}

    在20世纪初期和中期，化学工程学解决工程问题大都基于相似放大理论与实验归纳方法，在很多化工单元过程（流体传输、传热、干燥、蒸馏）中取得了很大的成功。

    但是，相似放大法不能简单地应用于化学反应器的模拟和设计。实际反应器中，质量传递、动量传递、热量传递与化学反应（\highlight{三传一反}）并存，不能单纯依靠化学动力学的知识来解决反应器的放大等相关问题。
\end{frame}


\begin{frame}
    \frametitle{工业反应器的放大}
    \begin{enumerate}
      \item 相似放大法
        \begin{itemize}
          \item 应用相似原理进行放大
          \item 有一定局限性，只适用于物理过程放大，不适用于化学过程的放大。
        \end{itemize}
      \item 逐级经验放大法
        \begin{itemize}
          \item 小型反应器进行工艺试验→规模稍大模型试验→中间试验→工业规模
          \item 不能深入研究过程的内在规律；
          \item 试验步骤由人为规定，并非科学合理的研究程序；
          \item 放大是根据试验结果外推，并不一定可靠。
        \end{itemize}
      \item 数学模型法
        \begin{itemize}
          \item 实验室研究化学反应特征→冷模试验研究传递过程特征→建立数学模型，预测工业反应器性能→中试检验模型的等效性→计算机试验进行检验和修正→工业反应器
          \item 可实现高倍数放大；缩短开发周期；减少人力物力消耗。
        \end{itemize}
    \end{enumerate}
\end{frame}
